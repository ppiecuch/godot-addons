## Vector Icons for Godot Engine

Perfect for buttons, logos and nav/tab bars. Easy to extend, style and integrate into your project.

This project is inspired by [react-native-vector-icons](https://github.com/oblador/react-native-vector-icons)

## Installation

1. Download this repo and rename the unzip folder name to `vector-icons`
2. Move the `vector-icons` folder into `res://addons` folder of your project
3. Enable the `VectorIcons` plugin in your project settings

## Q&A

### I don't need so much kind of icon set can I remove some of them ?

Of course yes. Just remove the `Iconset.ttf` and `Iconset.gd` file.
